# Base Converter

A CLI tool for converting numbers between binary, decimal and hexadecimal bases.

## Compile

gcc converter.c -o converter

## Run

./converter

## Usage

Select the origin and destiny base, then enter the number to convert.
The program validates the input according to the selected base.

## Supported conversions

- Binary → Decimal
- Binary → Hexadecimal
- Decimal → Binary
- Decimal → Hexadecimal
- Hexadecimal → Binary
- Hexadecimal → Decimal
